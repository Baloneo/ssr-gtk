## SSR LINUX GUI
简易版的在linux下的ssr图形化界面

* 订阅ssr地址生成制定配置文件
* 后台运行

## 安装使用
#### 安装
```
cd THIS_PROJECT
chmod +x ./install.sh
./install.sh
```
安装好之后的在你的软件列表能找到

#### 卸载
退出软件,执行下面命令
```
cd THIS_PROJECT
chmod +x ./uninstall.sh
./uninstall.sh
```
